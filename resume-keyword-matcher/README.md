# 简历关键词匹配分析工具

> 一个轻量级 CLI 工具，帮你分析简历与目标岗位的关键词匹配度。

## 功能

- 输入简历文本 + 岗位描述，计算关键词匹配得分
- 输出可视化匹配报告：匹配率、匹配关键词、缺失关键词
- 支持中英文混合文本
- 零外部依赖，纯 Python 实现

## 快速开始

```bash
python matcher.py --resume resume.txt --job job_desc.txt
```

或者直接在命令行输入：

```bash
python matcher.py
# 然后按提示粘贴文本
```

## 示例输出

```
岗位关键词总数: 12
简历匹配关键词: 8
匹配率: 66.7%

✅ 已匹配关键词:
  Python, 数据分析, MySQL, Linux, Git, Java, 团队协作, 文档编写

❌ 缺失关键词:
  微服务架构, Docker, Kafka, 分布式系统

💡 建议补充方向:
  - 微服务相关项目经验
  - Docker/K8s 容器化技能
  - 消息队列中间件
```

## 适合谁用

- 正在找工作的应届生 / 在校生
- 需要针对不同岗位优化简历的求职者
- 想快速评估自己与岗位匹配度的人
